## The Problem

As developers, we've all been there:
- Staring at legacy code wondering "what does this even do?"
- Debugging cryptic error messages for hours
- Context-switching to ChatGPT or Stack Overflow constantly
- Wishing we had a senior developer to explain things

## The Solution

FlowPilot brings the power of Large Language Models directly into your editor. No more context switching, no more copy-pasting code into ChatGPT.
